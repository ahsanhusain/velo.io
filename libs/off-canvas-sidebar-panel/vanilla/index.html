Events

You can fire the opening event by calling the open function: sidebar.open()

You can do the same for the closing event with sidebar.close()

Available options

You can use the JS section of this codepen to play with options.

  • selector: string;
  • triggerer: string;
  • mask: boolean;
  • align: "right" | "left";
  • top: string;
  • width: string;
  • gap: string;
  • opened: boolean;
  • easing: string;
  • zIndex: number;
